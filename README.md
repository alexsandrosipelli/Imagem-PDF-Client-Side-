# Imagem → PDF (Client-Side)

Ferramenta web para converter múltiplas imagens em um único arquivo PDF diretamente no navegador.

---

## Objetivo

Permitir a geração de PDF a partir de imagens sem necessidade de backend, servidor ou instalação de software.

---

## Funcionalidades

- Upload múltiplo de imagens
- Geração automática de PDF
- Download do arquivo final

---

## Tecnologias

- HTML5
- CSS3
- JavaScript (ES6+)
- jsPDF

---

## Estrutura
image-to-pdf/
├── index.html
├── style.css
└── script.js
---

## Execução

Abrir o `index.html` no navegador ou acessar via GitHub Pages https://alexsandrosipelli.github.io/Imagem-PDF-Client-Side-/.

---

## Observações

- Processamento 100% client-side
- Nenhum arquivo é enviado para servidor
